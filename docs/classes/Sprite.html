    Overview

    Sprite class

    Game Item displaying an asset or a portion of it in a DOM element on the stage Can be a child of an Entity, in which case it will wrap it's element in the Entity's wrap element, otherwise will render inside its layer

    The rendering makes use of the SpriteRenderer class It also makes use of decorators, which are other overlapped sprites that can be switched on an off to create effects

    Initialise with a 'type' option, which will be the key of its preset in www/game/sprites.json

    Read Game Data class for more on JSON assets
