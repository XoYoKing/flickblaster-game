    Class: BaseItem

    Defined in: src/game/items/BaseItem.coffee

    Overview

    Base Item class

    Item classes that interact with the game logic have to extend from here

    E.g. Entities, Bodies, Sprites and Walls extend from BaseItem

    The World and Layer classes will need to always have acces to the basic method specified in this class

    Direct Known Subclasses

    Body, Entity, Sprite
